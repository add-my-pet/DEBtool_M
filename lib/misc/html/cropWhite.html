cropWhite

removes white borders and makr transparent in png image

Syntax

cropWhite (imgIn,imgOut,transparent)

Description

Removes white borders in png image and make white transparent; Transparency requires program magick (with path to it); see https://imagemagick.org/script/download.php

Input:

  • imgIn: string with name of png input image, not containing '.'
  • imgOut: optional string with name of png output image (default: imgIn)
  • transparent: optional boolean to replace white by transparent (default 1)

Output:

  • no explicit output, but output file imgOut.png is written
