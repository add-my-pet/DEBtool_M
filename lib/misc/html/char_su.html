char_su

characteristic value based on minimazation of loss function F_su

Contents

  • Syntax
  • Description
  • Example of use

Syntax

val = char_su (data)

Description

computes a characteristic value based on minimazation of loss function F_su. Loss function F_su is defined as F_Su = (val - data).^2 ./ (val.^2 + data.^2)' * weights. The minimum is found from d/d val F_su = 0

Input:

  • data: (n,1 or 2)-array of data values with optional weights in second column (default ones)

Output:

  • val: characteritic value (i.e. a kind of mean or median)

Example of use

x=normrnd(12,5,[10,1]); [mean(x) median(x) char_su(x)]
