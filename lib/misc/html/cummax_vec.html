cummax_vec

cumulative maxima of a sequence of numbers

Contents

  • Syntax
  • Description
  • Example

Syntax

xmax = cummax_vec(x)

Description

computes cumulative maxima of a sequence of numbers

Input

  • x: n-vector

Output:

  • xmax: n-vector with cumulative maxima of x

Example

cummax_vec([1 3 6 4 7 6])


Published with MATLAB® R2015b
