char_sb

characteristic value based on minimazation of loss function F_sb

Contents

  • Syntax
  • Description
  • Example of use

Syntax

[val, info] = char_sb (data)

Description

computes a characteristic value based on minimazation of loss function F_sb. Loss function F_sb is defined as F_sb = (val - data).^2 ./ (val.^2 + data.^2)' * weights. The minimum is found from d/d val F_sb = 0

Input:

  • data: (n,1 or 2)-array of data values with optional weights in second column (default ones)

Output:

  • val: characteritic value (i.e. a kind of mean or median)
  • info: scalar with success (1) or failure (0)

Example of use

x=randN(12); [mean(x) median(x) char_sb(x)]
