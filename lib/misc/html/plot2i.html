plot2i

plot points in 2D where markers texts are specified of each point

Contents

  • Syntax
  • Description
  • Example of use

Syntax

[Hfig, Hleg] = plot2i (data, legend, Title)

Description

plot data in 2D, using data-point-specific markers or texts. For markers, the legend should specify, 'MarkerType', 'MarkerSize', 'LineWidth', 'MarkerFaceColor', 'MarkerEdgeColor'. For text, the legend should specify, 'String', 'FontSize' (default 10), 'FontColor' (default black).

Input:

  • data: (n,2)-array with data points
  • legend: (n,2)-array with legend for markers or texts
  • Title: optional string with title

Output:

  • no output

Example of use

plot2i([1 1.2 2; 2 3.1 3]', {{'bla';'?';'str'},8*ones(3,1),ones(3,1)*[0 0 1]}, 'ref2026')
