get_d_V

Gets specific density of structure from phylum or class

Contents

  • Syntax
  • Description
  • Remarks

Syntax

d_V = get_d_V

Description

Sets specific density d_V according to taxonomic classification

Input

  • phylum: name of animal phylum (one of 36 possibilities)
  • class: name of class, only used for chordates

Output

d_V: scalar with specific density in g/cm^3 of dry mass

Remarks

d_V has units g/cm^3 and refers to dry mass. Since the specific density of wet mass is taken to be 1 g/cm^3, it can also be considered as a dry/wet mass ratio


Published with MATLAB® 7.14
