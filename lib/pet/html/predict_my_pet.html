Syntax

[Prd_data, info] = predict_my_pet(par, chem, data)

Description

Obtains predictions, using parameters and data

Input

  • par: structure with parameters (see below)
  • chem: structure with biochemical parameters
  • data: structure with data (not all elements are used)

Output

  • Prd_data: structure with predicted values for data
  • info: identified for correct setting of predictions (see remarks)

Remarks

Template for use in add_my_pet. The code is symplified by calling parscomp_st to compute scaled quantities. With the use of filters, setting info = 0, Pred_dat = {}, return, has the effect that the parameter-combination is not selected for finding the best-fitting combination: it acts as costumized filter.
