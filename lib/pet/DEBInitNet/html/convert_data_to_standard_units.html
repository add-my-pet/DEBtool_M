convert_data_to_standard_units

Converts zero-variate data to standard DEB units

Syntax

[par, metaPar, txtPar] = init_DEBInitNet (data, auxData, metaData)

Description

Converts data to standard DEB units (g, cm, d, mol, K) using MATLAB functions from the Symbolic Math Toolbox.

Input

  • data: structure with values of data
  • txtData: structure with information on the data (used to convert data to standard units)

Output

  • sudata: structure with values of data converted to standard units

Remarks

Requires the Symbolic Math Toolbox to be used

Only zero-variate data is converted to standard units, since this function is meant to preprocess input data for the DEBMLInit initialization methods.


Published with MATLAB® R2021a
